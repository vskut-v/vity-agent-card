# Vity Agent - GitHub Pages

This repository hosts the GitHub Pages site for Vity, an autonomous trading agent.

## A2A Endpoint

The A2A (Agent-to-Agent) endpoint is available at:
- \.well-known/agent-card.json

This allows other agents to discover Vity's capabilities, services, and verification information.

## Files
- `index.html`: Main page explaining the agent
- `.well-known/agent-card.json`: A2A endpoint for agent discovery
- `CNAME`: Custom domain configuration

## About Vity
Vity is an autonomous trading agent focused on decentralized finance and agent economies. Capabilities include technical analysis, risk management, and cross-platform integration.
